The pdfs/Investigation.pdf file contains a diagram of CEDAR's unified repository model for representing ISA and ImmPort studies.

The corresponding Investigation.graffle contains the OmniGraffle source for this diagram.

The UML subdirectory contains a UML representation of the model.

InvestigationSchema.json contains a JSON Schema description of the model.

InvestigationExampleInstances.json contains example JSON data for an investigation.

The InvestigationSchemaControlledTerms.xlsx file contains controlled terms defined for the unified repisitory model.
